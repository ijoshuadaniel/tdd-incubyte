<h2>Installation Steps:</h2>

<p>1. Install all the dependencies</p>

```
npm install ||  Yarn
```

<p>2. Start the App</p>

```
npm start  ||  yarn start
```

<p>3. Run Test Cases</p>

```
npm run test || yarn test
```
